How to run:
	1. Load the hmintz_481_HW1 visual studio project
	2. enure that you are using the release configuration
	3. ensure that the prooper sfml libraries are loaded
	4. Run the source.cpp


How to control character:
	- The character is the green circle
	- You can  accelerate the circle left or right using the left/right arrow keys
	- You can make the circle jump by pressing space
